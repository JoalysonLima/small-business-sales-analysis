# Small Business Sales Data Cleaning & Analysis

## Overview

This project simulates a real-world data analysis task for a small business.

The goal is to clean, organize, analyze, and present sales data in a way that helps a business owner understand performance, identify opportunities, and make better decisions.

## Business Context

The fictional business sells products through different sales channels, including physical store, online store, and marketplace. The original sales data contains common issues such as missing values, duplicates, inconsistent formatting, and incorrect data types.

## Business Questions

This project aims to answer the following questions:

- What is the total revenue?
- How many orders were placed?
- What is the average order value?
- Which products generated the most revenue?
- Which categories performed best?
- Which sales channel had the highest revenue?
- How did sales perform over time?
- Are there any data quality issues that need attention?
